AgendaOffice2000 v1.2 - Instrucciones de uso (Windows 10)
------------------------------------------------------

Ruta de ejemplo donde podes extraer el proyecto:
D:\Usuario\Desktop\AgendaOffice2000

Pasos para ejecutar:

1) Abrí PowerShell o Símbolo del sistema (CMD).
2) Entrá a la carpeta del proyecto:
   cd D:\Usuario\Desktop\AgendaOffice2000_v1.2
3) Instalá dependencias:
   npm install
4) Iniciá la app:
   npm start

- El servidor de desarrollo abrirá la app en http://localhost:3000
- Los datos (clientes y servicios) se guardan en localStorage en cada equipo.
- Si querés que dos equipos compartan los datos, puedo integrar Firebase o un backend.
